# B站数据可视化

## 概述

- 该项目目前【未完成】。
- 项目主要是使用python对B站上的一些番剧、视频等数据进行爬取（数据约1w+），并对获取到的数据进行过滤，整合。再通过Vue.js对页面进行展示，对筛选出的数据使用Echarts来生成可视化图表。
- 界面参考自taptap。

## 技术栈

vue2 + vue-router + vue-cli+ echarts + node.js+ express+ python
